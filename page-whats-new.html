Hybrid Cloud Platform 3.08

Remote Update of Secondary Gateways

Remote Update of Secondary Gateways

  • HCP 3.9 includes a new way to manage Secondary Gateways in your HCP infrastructure – you can now remotely update Secondary Gateways from the Administrator Web User Interface.
  • On actioning updates, the latest version is applied to selected servers and checks made to ensure that the process is successfully completed.
  • Should you need to, there is Rollback option and with the new Servers view, you can easily see status, version and last configuration of your HCP infrastructure.
  • This will save time and effort, enabling customers to easily keep their complete HCP environment updated with the latest features.

For a full list of changes, view the version history in the Partner Zone.

How to upgrade

Upgrade an existing version of EveryonePrint Hybrid Cloud Platform (HCP) simply by running the new installer on top of your old installation, and it will upgrade as needed. You can upgrade from any previous version directly to latest. All configurations, license keys etc will be retained.

1
2

If you run a Primary HCP server, make sure to always upgrade the primary server first, before upgrading any gateways (”secondary servers”).

The EveryonePrint HCP primary cloud servers have already been upgraded.
